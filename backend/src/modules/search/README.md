# Search Bounded Context

The Search bounded context is responsible for indexing, querying, and retrieving knowledge across the Connect-Africa platform.

## Features
- Full-text and hybrid search (PostgreSQL + AI)
- Faceted search
- Graph-based knowledge retrieval
- Ontology-aware filtering
- Autocomplete and trending suggestions

## Architecture
- Presentation (Interfaces)
- Application (Query Handlers)
- Domain (Repositories/Services)
- Infrastructure (Postgres/Semantic Providers)
