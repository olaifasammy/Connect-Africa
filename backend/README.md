# Connect Africa Backend - Identity Context

The Identity context provides robust authentication, session management, and authorization (RBAC) capabilities.

## Architecture
- **DDD:** Aggregates, Value Objects, Domain Policies.
- **Clean Architecture:** Domain and Application layers isolated from Infrastructure.
- **Infrastructure:** PostgreSQL, Redis, Bcrypt, JWT.
