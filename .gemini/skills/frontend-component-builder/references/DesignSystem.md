# Design System Reference

When building UI components, adhere to these standards:

- **Spacing**: Use standard spacing variables (check `frontend/src/shared/styles/spacing.ts`).
- **Typography**: Use standard typography styles (check `frontend/src/shared/styles/typography.ts`).
- **Colors**: Use variables from the theme (check `frontend/src/shared/styles/theme.ts`).
- **Components**: Always prefer `shared/` components for standard elements:
    - `Button`
    - `Modal`
    - `Avatar`
    - `Table`
    - `LoadingSpinner`
    - `Toast`
